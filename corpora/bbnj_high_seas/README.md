# Corpus module: `bbnj_high_seas`

This folder is a **pointer**, not the corpus. It contains `source_pointer.json` (canonical
locations + inclusion status) and `sample_manifest.json` (a small English-only sample for
demonstration/tests).

**The full corpus is not here.** Get it from the canonical sources:

- Repository: https://github.com/dacheah/bbnj-high-seas-treaty-corpus
- Dataset: https://huggingface.co/datasets/dacheah/bbnj-high-seas-treaty-corpus
- Browsable site: https://dacheah.github.io/bbnj-high-seas-treaty-corpus/

The full corpus holds the 2023 BBNJ Agreement in all six authentic UN languages; this sample ships
English only. Content hashes returned for this sample are computed live from the shipped excerpts.
