# Corpus module: `deep_seabed_mining`

This folder is a **pointer**, not the corpus. It contains `source_pointer.json` (canonical
locations + inclusion status) and `sample_manifest.json` (a small sample for demonstration/tests).

**The full corpus is not here.** Get it from the canonical sources:

- Repository: https://github.com/dacheah/deep-seabed-mining-law-corpus
- Dataset: https://huggingface.co/datasets/dacheah/deep-seabed-mining-law-corpus
- Browsable site: https://dacheah.github.io/deep-seabed-mining-law-corpus/

Content hashes returned for this sample are computed live from the shipped excerpts and are fully
reproducible.
